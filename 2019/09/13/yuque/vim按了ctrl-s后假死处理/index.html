vim按了ctrl-s后假死处理
发表于 2019-09-14|LinuxSoftware
字数总计:190|阅读时长: 1 分钟

理论上这不能算上一篇文章的,但是写代码过头了,经常在 vim 里面按了 Ctrl-S ,导致无法向终端输入.

这篇文章当做是让自己记住这种情况怎么处理吧

如果按了 Ctrl-S  的话,这时候终端就展现出一种卡死的状态,无论输入什么都没有效果.但其实这个时候终端并没有卡死,而是停止输出了.只需要退出这种状态即可.

退出这种状态也很简单,按下 Ctrl-Q ,即可退出停止输出.

下次就不要按了 Ctrl-S  然后用重新使用 SSH 连接这种方式解决了,有时候 vim 写了一大板就很痛苦了.
